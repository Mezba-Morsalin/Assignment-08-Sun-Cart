# 🌞 Sun Cart

**Sun Cart** is a modern e-commerce web application focused on **summer accessories**. It provides a smooth and responsive shopping experience with authentication, dynamic UI, and interactive components.

---

## 🚀 Live Features

- 🛍️ Browse summer accessories products  
- 🔐 User authentication (Sign Up / Login / Profile Update)  
- 👤 User profile management (name, email, image, password)  
- ⚡ Fast and responsive UI  
- 🎞️ Smooth animations and transitions  
- 🔔 Toast notifications for user actions  
- 📱 Fully responsive design  

---

## 🧰 Tech Stack

### Frontend
- ⚛️ React.js  
- ⚡ Next.js (App Router)  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 🎞️ Animate.css  
- 🌀 Swiper.js  
- 📢 React Hot Toast  
- 🔁 React Marquee  
- 🧠 React Hook Form  

### Backend & Auth
- 🔐 Better Auth  
- 🗄️ MongoDB  

---

## 📁 Project Structure

```
src/
 ├── app/
 │   ├── (auth)/        # Authentication pages (signin, signup)
 │   ├── (main)/        # Main routes (home, products, profile, etc.)
 │   ├── api/           # API routes
 │
 ├── components/        # Reusable components
 ├── shared/            # Navbar, Footer, etc.
 ├── lib/               # Auth client, DB config
 ├── assets/            # Images & static files
```

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Mezba-Morsalin/Assignment-08-Sun-Cart

# Go to project folder
cd sun-cart

# Install dependencies
npm install

# Run development server
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file and add:

```
MONGODB_URI=your_mongodb_connection_string
BETTER_AUTH_SECRET=your_secret_key
```
